Why we built this

Every developer has been there — you need to quickly format some JSON, decode a JWT, or check a colour contrast ratio, and you end up on a random site that makes you sign up, pastes ads everywhere, or — worse — uploads your data to a server you don't control.

tinybench.dev exists to fix that. Every tool runs client-side in your browser using JavaScript. Your data never leaves your machine. No accounts, no tracking, no nonsense.

Tech stack

tinybench.dev is a pure static site — plain HTML, CSS, and vanilla JavaScript. No frameworks, no build step, no dependencies. Hosted on Cloudflare Pages with automatic deploys from GitHub. The source code is open on GitHub.
